## 二、核心架构升级

### 1. 双版本 MoE 架构

| 版本 | 总参数 | 激活参数 | 定位 |
|------|--------|----------|------|
| **V4-Pro** | 1.6T | 49B | 极致性能，复杂推理与专业代码 |
| **V4-Flash** | 284B | 13B | 高性价比，快速响应 |

> MoE 的核心思想：每次推理只激活部分专家参数，兼顾性能与效率。

### 2. 百万级超长上下文

- **技术方案**：DSA 稀疏注意力（DeepSeek Sparse Attention）+ Token 维度压缩
- **意义**：打破长文本处理的算力与显存壁垒，让百万上下文从"高端专属"变为"全民可用"
- **应用场景**：百万字文档理解、长篇报告分析、多轮对话历史记忆

### 3. 底层架构创新

#### 3.1 Manifold-Constrained Hyper-Connections (mHC)

- **发布时间**：2026 年 1 月论文披露
- **核心作用**：改变信息在模型层间的流动方式，使深层表征能被更高效地利用
- **意义**：V4 架构基石之一

#### 3.2 Engram 条件记忆技术

- **核心作用**：让模型能够**选择性地记忆和调用上下文信息**
- **意义**：支持超过 100 万 token 长上下文的关键技术支撑

### 4. 原生多模态能力

- V4 是 DeepSeek 首款**原生多模态大模型**
- 同时理解和生成**图片、视频、文本**
- 补齐了过往仅擅长文字的短板

### 5. 国产芯片深度适配（战略转向）

- **绕过英伟达**，优先与国产芯片厂商合作深度调优
- 在**推理阶段**实现国产芯片全面适配
- 标志着中国 AI 从"用别人的芯片跑自己的模型"走向"用自己的芯片跑自己的模型"

## 五、学习心得

1. **架构效率优先**：DeepSeek 的核心竞争力不是堆算力，而是通过 MoE + 稀疏注意力 + mHC 等架构创新实现效率突破
2. **长上下文普惠化**：DSA 稀疏注意力 + Token 压缩是关键，将计算复杂度从 O(n²) 大幅降低
3. **国产化战略意义重大**：推理阶段的国产芯片适配是大模型商业化落地的核心环节
4. **开源即竞争力**：MIT 开源协议 + 极低 API 定价，构建生态护城河
